## Machine Problem 14

Create a program that will detect if the internet connection is fast or slow where:

N<=85 = FAST
N>85 = SLOW

Include the average of the internet connection and add if it is fast or slow.

Input: Each input should be an integer.

Output: The output should be either FAST or SLOW per input. Include the average as an integer and declare if it is FAST or SLOW.

Sample Input: 

```
134
56
78
90
100
123
```

Sample Output: 

```
FAST
SLOW
SLOW
FAST
FAST
FAST
96
FAST
```
