## Machine Problem 2

You are given a list of items, then you are tasked to remove the outlier of the list. Write a program that will count how many outliers were removed and how many items are left in the list.

Input: The first line of input should be a list of strings spacing each. The next input should be the outlier.

Output: The first line of output should be the number of outliers found. The next output should be the number of items in the list left.

Sample input:
```
apple orange banana orange apple grape guava
orange
```

Sample Output:
```
2
5
```
