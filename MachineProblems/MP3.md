## Machine Problem 3

You are given a string containing parentheses symbols. Write a program that will determine whether the parentheses in the string are balanced.

Input: The first line of input should be a string of parenthesis. If there is nothing, the program should print "PLEASE ENTER A VALID STRING" and asks for the input again.

Output: The first line of output should be either "BALANCED" or "NOT BALANCED". If the the parenthesis are balanced, it should print "BALANCED". If the input contains a mismatch or invalid character, it should print "NOT BALANCED".

Sample input:
```
{([])}
```

Sample Output:
```
BALANCED
```
