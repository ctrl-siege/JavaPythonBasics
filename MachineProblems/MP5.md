## Machine Problem 5

You are tasked to create a Christmas tree using asterisks (*). The tree has a leafy top section made of layers and a trunk at the bottom.

Input: The first input should be the positive number of rows of leaves. The second input should be the positive number of layers of the tree. Each successive layer grows in size. The third input should be the width of the trunk. The fourth input should be the height of the trunk. Any input that is 0 or below should print "INVALID INPUT."

Output: The output should be the created Christmas tree. Centering is optional.

Sample input:
```
4
3
2
2
```

Sample output:
```
      *
     ***
    *****
   *******
      *
     ***
    *****
   *******
  *********
      *
     ***
    *****
   *******
  *********
 ***********
      **
      **
```
