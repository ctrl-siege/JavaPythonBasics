## Machine Problem 7

Create a program that runs the Fibonacci sequence (0, 1, 1, 2, 3, 5...)

Input: The program accepts an input that exists in the Fibonacci sequence. 

Output: If the number exists in the sequence it displays the placement of the number (counting starts at 0). If the number does not exist in the sequence, it displays "Does not exist".

Sample input:
```
8
```

Sample output:
```
8 exists at position 6
```
