## Machine Problem 1

You are given a starting budget and will spend money by buying items one at a time. Your program must keep track of how many items were bought and how much money remains.

Input: The first line of input should be a positive number that is greater than 10, other it should print "Invalid" and exit the program. The remaining lines of input should be a positive number. If the input exceeds the current budget or the input is a 0, then the input should be stopped. Negative numbers should be ignored.

Output: The first line should be how many items you have spend. The second line should be your remaining budget.

Sample input:
```
45
15
10
-2
0
```

Sample Output:
```
2
20
```
