## Machine Problem 8

There are 5 candidates. Allow 20 voters to vote by entering a number (1–5). Display the following: 

- Total votes per candidate.
- Display the winner.

If tie occurs, print: "Tie detected".


Sample input:
```
1
1
2
3
1
4
5
1
2
3
1
4
1
5
1
2
1
3
4
1
```

Sample output:
```
Candidate 1: 9 votes
Candidate 2: 3 votes
Candidate 3: 3 votes
Candidate 4: 3 votes
Candidate 5: 2 votes

Winner: Candidate 1
```
